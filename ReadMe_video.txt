Распределённые вычисления, многопоточность, асинхронность и мультипроцессорность в python


Виды нагрузок:
    CPU bound - нагрузка на процессор
    I/O bound - процессор ожидает операции ввода/вывода


Виды вычислений:
    многопроцессорность - своя область памяти у каждого процесса
    многопоточность - общая память между потоками, блокировки исполнения GIL в зависимости от вида нагрузки
    асинхронность - конкурентные потоки
